


Install Brew (/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)")
Install Node.js
Install watchman (brew install watchman)
Instal XCode
Install cocoapods (brew install cocoapods)
Install Android Studio
Set Android environment variables (
  export ANDROID_HOME=$HOME/Library/Android/sdk
  export PATH=$PATH:$ANDROID_HOME/emulator
  export PATH=$PATH:$ANDROID_HOME/tools
  export PATH=$PATH:$ANDROID_HOME/tools/bin
  export PATH=$PATH:$ANDROID_HOME/platform-tools
)

Create new project (npx @react-native-community/cli@latest init NewProject)
Open the project folder (cd NewProject)
Run it (npm run ios) (npm run android)


Error with XCode?
Update developer directory path (sudo xcode-select -s /Applications/Xcode.app/Contents/Developer)
Accept XCode licence (sudo xcodebuild -license accept)
Run (npm run ios)

Error with Android?
Navigate to /NewProject/android/
Create file local.properties
Add "sdk.dir=/Users/lmadrigal/Library/Android/sdk" into that file and save


Still error with Android?

Generate Z shell configuration (
  cat << 'EOF' >> ~/.zshrc
  export ANDROID_HOME=$HOME/Library/Android/sdk
  export PATH=$PATH:$ANDROID_HOME/emulator
  export PATH=$PATH:$ANDROID_HOME/platform-tools
  EOF
)

Reload terminal (source ~/.zshrc)

Still still error with Android?

Install JDK 17 (brew install openjdk@17)
Symlink for mac to recognize the sdk (sudo ln -sfn /opt/homebrew/opt/openjdk@17/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-17.jdk)

clean project and build again (cd android && ./gradlew clean && cd ..)
npm run android

Still still still error with Anndroid?

Set JAVA_HOME 
(echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 17)' >> ~/.zshrc)
(source ~/.zshrc)

Clean and build again (cd android && ./gradlew --stop && cd ..)

Run it(npm run android)
